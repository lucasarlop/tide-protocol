---
description: Mostra detalhes de uma Wave.
agent: tide-steward
---

Mostre os detalhes da Wave `$ARGUMENTS`.

Use:

```bash
tide wave show $ARGUMENTS
```

Se o CLI não estiver disponível, leia `.opencode/waves/$ARGUMENTS/wave.md`.

Inclua status, intenção, fronteira, baseline, evidência, riscos e próximos passos seguros.
